**Структура абстрактного синтаксического дерева (AST)**
Семантический модуль оперирует следующими типами узлов:

ProgramNode — корневой узел дерева. Содержит список всех корректных инструкций программы (Nodes).
VarDeclNode — узел объявления и инициализации переменной. Хранит тип (TypeName), имя (Name) и ссылку на узел инициализатора.
WhileNode — узел цикла while. Хранит узел условия (Condition) и список инструкций внутри тела цикла (Body).
BinaryOpNode — узел бинарной операции сравнения. Хранит оператор (<, >, ==) и ссылки на левую и правую части.
UpdateNode — узел изменения переменной (инкремент/декремент). Хранит имя переменной и оператор (++, --).
LiteralNode / IdNode — листовые узлы, представляющие конкретное значение (константу) или идентификатор переменной.

**Формат графического вывода AST в программе**
При вводе корректного кода:
```
Int i = 0;
while (i < 10) {
    i++;
};
```
Программа сформирует следующее дерево в консоли вывода:
```
Program
├── Assignment (id: i, type: Int)
│   └── Literal (Целочисленная константа: 0)
└── WhileLoop
    ├── Condition (operator: <)
    │   ├── Identifier (имя: i)
    │   └── Literal (Целочисленная константа: 10)
    └── ExpressionStatement (id: i, op: ++)
```
