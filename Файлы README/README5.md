Семантический модуль оперирует следующими типами узлов:
1. Root (Корневой уровень)
Тип узла: ProgramNode
Содержимое (Nodes): Список, содержащий один узел типа WhileNode.

2. Узел цикла (WhileNode)
Condition: Ссылка на узел BinaryOpNode (соответствует ветке condition на схеме).
Body: Список инструкций, содержащий один узел UpdateNode (соответствует ветке Stmnt block).

3. Узел условия (BinaryOpNode)
Оператор: < (из ветки Comp operator)
Левая часть (Left): IdNode
Имя (Name): "i" (из ветки variable)
Правая часть (Right): LiteralNode
Значение (Value): 10 (из ветки number)

**Формат графического вывода AST в программе**
При вводе корректного кода:
```
Int i = 0;
while (i < 10) {
    i++;
};
```
Программа сформирует следующее дерево в консоли вывода:
```
Program
├── Assignment (id: i, type: Int)
│   └── Literal (Целочисленная константа: 0)
└── WhileLoop
    ├── Condition (operator: <)
    │   ├── Identifier (имя: i)
    │   └── Literal (Целочисленная константа: 10)
    └── ExpressionStatement (id: i, op: ++)
```
