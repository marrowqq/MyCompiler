1. ProgramNode (Корень)
Это самый верхний узел, представляющий всю программу или текущую область видимости.
Свойства:
Nodes: List<WhileNode> — список, содержащий одну инструкцию (наш цикл).

2. WhileNode (Узел цикла)
Соответствует конструкции while (condition) { Body };.
Свойства:
Condition: BinaryOpNode — ссылка на узел бинарной операции сравнения.
Body: List<UpdateNode> — список инструкций внутри фигурных скобок {} (блок Stmnt block на вашей схеме).

3. BinaryOpNode (Узел условия)
Отвечает за логическое выражение i < 10 в круглых скобках.
Свойства:
Operator: String = "<" — оператор сравнения.
Left: IdNode — левый операнд (переменная).
Right: LiteralNode — правый операнд (константа/число).

4. IdNode (Идентификатор переменной)
Листовой узел для переменной i внутри условия.
Свойства:
Name: String = "i" — имя переменной.

6. LiteralNode (Литерал/Константа)
Листовой узел для фиксированного числа 10 внутри условия.
Свойства:
Value: Int = 10 — конкретное числовое значение.

7. UpdateNode (Узел изменения переменной)
Соответствует инструкции инкремента i++ внутри тела цикла.
Свойства:
Name: String = "i" — имя изменяемой переменной.
Operator: String = "++" — оператор изменения (выбран из ветки Increment stmnt на схеме).

**Формат графического вывода AST в программе**
При вводе корректного кода:
```
Int i = 0;
while (i < 10) {
    i++;
};
```
Программа сформирует следующее дерево в консоли вывода:
```
Program
├── Assignment (id: i, type: Int)
│   └── Literal (Целочисленная константа: 0)
└── WhileLoop
    ├── Condition (operator: <)
    │   ├── Identifier (имя: i)
    │   └── Literal (Целочисленная константа: 10)
    └── ExpressionStatement (id: i, op: ++)
```
