Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

	3.Описание проекта

Данное приложение представляет собой специализированный текстовый редактор, разработанный как основа для будущего языкового процессора.
Редактор предоставляет все необходимые функции для работы с текстом, а также содержит специальную область для вывода результатов анализа кода.

	4.Используемые технологии

Язык программирования: C#
Фреймворк для GUI: Windows Forms (WinForms)
Платформа: .NET 8.0
Среда разработки: Microsoft Visual Studio 2022
Система контроля версий: Git

	5.Инструкция по сборке и запуску

Требования для сборки
Установленный [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
(Опционально) Visual Studio 2022 или другая IDE

Через командную строку:

Клонируем репозиторий
git clone https://github.com/marrowqq/MyCompiler.git
cd MyCompiler

Собираем проект
dotnet build -c Release

Создаём исполняемый файл
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
