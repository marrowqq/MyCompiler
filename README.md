ЗАДАЧА 13: Построить РВ для того, чтобы найти все годы между 2000 и 2010.

Регулярное выражение:
\b(200[0-9]|2010)\b

Пояснение обозначений:
\b     - граница слова
(      - начало группы
200    - literal "200"
[0-9]  - любая цифра от 0 до 9
|      - оператор "или"
2010   - literal "2010"
)      - конец группы
\b     - граница слова

Примеры строк, которые должны находиться:
2000
2001
2005
2009
2010
В 2005 году был запуск
Годы: 2000, 2001, 2008

Примеры строк, которые не должны находиться:
1999
2011 
20000
200 
20 05
2000й


ЗАДАЧА 9: Номера карт платежной системы Мир

Регулярное выражение:
\b220[0-4]\d{12}\b

Пояснение обозначений:
\b       - граница слова 
220      - первые три цифры номера карты Мир
[0-4]    - четвертая цифра от 0 до 4
\d       - любая цифра
{12}     - ровно 12 повторений предыдущего символа 
\b       - граница слова

Примеры строк, которые должны находиться:
2200123456789012
2201123456789012
2202123456789012
2203123456789012
2204123456789012
Номер карты: 2200123456789012

Примеры строк, которые не должны находиться:
2205123456789012 
220012345678901 
22001234567890123 
220012345678901a
3200123456789012
2200-1234-5678-9012
22001234567890


ЗАДАЧА 3: IPv6 адрес с префиксом

Регулярное выражение:
\b([0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}/\d{1,3}\b

Пояснение обозначений:
\b                - граница слова
(                 - начало группы
[0-9a-fA-F]       - шестнадцатеричная цифра
{1,4}             - от 1 до 4 шестнадцатеричных цифр
:                 - двоеточие
)                 - конец группы
{7}               - ровно 7 повторений группы
[0-9a-fA-F]{1,4}  - последняя восьмая группа 
/                 - слеш
\d                - любая цифра
{1,3}             - от 1 до 3 цифр
\b                - граница слова

Примеры строк, которые должны находиться:
2001:0db8:85a3:0000:0000:8a2e:0370:7334/64
2001:db8:85a3:0:0:8a2e:370:7334/48
FF01:0:0:0:0:0:0:101/112

Примеры строк, которые не должны находиться:
2001:0db8:85a3:0000:0000:8a2e:0370:7334 
2001:0db8:85a3:0000:0000:8a2e:0370:7334/ 
2001:0db8:85a3:0000:0000:8a2e:0370:7334/1280 
2001:0db8:85a3:0000:0000:8a2e:0370/64
192.168.1.1/24 
2001:0db8:85a3:0000:0000:8a2e:0370:7334/abc
